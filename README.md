# 🎨 Color Vision Deficiency Simulator

A web application that allows users to upload an image and preview how it appears under different types of color vision deficiencies. The application helps designers, developers, and students understand how images are perceived by people with different forms of color blindness.

## ✨ Features

* Upload images (JPG, PNG, WebP)
* Preview the original image
* Simulate:

  * Protanopia
  * Deuteranopia
  * Tritanopia
  * Achromatopsia
* Switch between simulation modes instantly
* Side-by-side comparison of the original and simulated image
* Responsive and user-friendly interface

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript

## 🚀 How to Use

1. Open the application.
2. Upload an image from your device or use the sample image.
3. Select a color vision deficiency mode.
4. Compare the original image with the simulated version.

## 📂 Project Structure

```
index.html
```

*(The project is built in a single HTML file containing HTML, CSS, and JavaScript.)*

## 🎯 Purpose

The goal of this project is to raise awareness about color accessibility and demonstrate how digital content appears to users with different types of color vision deficiencies. It can help improve inclusive and accessible design practices.
